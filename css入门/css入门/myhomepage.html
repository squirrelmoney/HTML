第一次网页设计

到底什么是网页设计呢？

百度上是这样解释的
        网站设计，要能充分吸引访问者的注意力，让访问者产生视觉上的愉悦感。因此在网页创作的时候就必须将网站的整体设计与网页设计的相关原理紧密结合起来。网站设计是将策划案中的内容、网站的主题模式，以及结合自己的认识通过艺术的手法表现出来；而网页制作通常就是将网页设计师所设计出来的设计稿，按照W3C规范用html将其制作成网页格式。

而在我看来

        网页设计就像一张白纸，你在上面画出你想要的板块模式、排版，而对于我这个初学者而言，排版还没能掌控，美观就别论了。这是我第一次做网页，我打算一切从简，重点着手排版，不会有过多装饰，太过花俏的现在还hold不住，越多东西越凌乱，我还是算了。
